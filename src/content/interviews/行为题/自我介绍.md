# 三分钟自我介绍

## 回答框架
建议固定成三段：

1. 我是谁，现在在做什么。
2. 我做过什么，重点突出与岗位最相关的经历。
3. 我为什么想来，以及我能带来什么。

## 示例表达
大家好，我是一名前端工程师，最近主要在做 React 与移动端 Web 体验优化。过去两年里，我负责过中后台系统和内容型产品，从组件抽象、页面性能到工程化都有比较完整的实践。

## 注意点
- 不要从大学经历一路讲到现在。
- 不要把简历逐条念一遍。
- 结尾一定要收束到“为什么适合这个岗位”。

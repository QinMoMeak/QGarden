    2. （2）宣传公安机关的有关工作情况；
    3. （3）发布治安警情、资讯；
    4. （4）推广群众性治安组织和个人的典型经验、先进事迹及示范作用；
2. **2、宣传群众的要求**：
    
    1. （1）宣传内容上要“三贴近”

    2. （2）宣传的方法要灵活多样：由于不同的地区、不同的单位以及不同的群体，在安全需求上具有不同的特点，在宣传方法选择上应该做到灵活多样。

        1. ①要求：
        2. ②形式：通过公益广告、网络宣传、QQ群、博客、微博、微信等现代媒体进行宣传。通过组织各种社会活动，如音乐会、体育活动的形式进行宣传。
3. **3、宣传方式**：
    
    1. （1）媒体宣传：分为运用传统媒体进行宣传和运用网络媒体进行宣传两种模式。比如传统媒体有社区讲座/培训等等，网络媒体有公益广告、网络宣传、Q0 群、博客、微博、微信等
    2. （2）广告宣传：常见的形式是发放宣传单、展板宣传、拉横幅、动态屏幕等。
    3. （3）口头宣传：分为报告、座谈会、讲座、法制课堂等四种形式。
        1. ①报告：主要有工作报告、动员报告、辅导报告等，是公安机关向群众报告工作成绩活动情况、动员群众支持参与社会治安活动的一种方式。

    4. （4）文艺宣传：通过组织各种社会活动，音乐会、体育活动等形式进行宣传。比如禁毒题材话剧《毒战》展示毒品危害。

### 组织群众

> 组织群众，是指公安机关根据公安群众工作的任务、目标，通过宣传、服务等措施争取群众的信任和支持，将群众组织起来，自觉抵御和防止社会失范行为的社会活动。

1. **1、社会治安防范组织的类型**：
    
    1. （1）‌自治性治安社会组织：这类组织通常由社区居民自发组织，负责社区内的治安维护工作。例如，治保会、社区巡逻队。
        
    2. （2）‌商业性治安社会组织：这类组织主要由商业机构设立，旨在保护其财产和员工安全。例如，保安服务公司、物业服务公司‌。
        
    3. （3）‌单位内部保卫组织：这类组织通常设立在企事业单位内部，负责本单位的治安保卫工作。例如，保卫处‌。
        
    4. （4）‌义务性治安社会组织：这类组织由志愿者组成，旨在通过义务劳动来维护社会治安。例如，义务反扒队、平安义警队、治安志愿队。
        
2. **2、组织动员的方式**：
    
    1. （1）动员和指导社区参与治安工作：公安机关发动和指导社区参与社会治安工作，主要通过开展警民共建安全文明社区活动，以及在社区各类管理机构中发挥组织、督导作用等方式来实现。
        
    2. （2）动员和指导单位参与治安工作：发动、组织企事业单位的群众性治安组织参与治安防范工作，主要是坚持预防为主、单位负责、突出重点、保障安全的方针，通过依法指导、监督企事业单位自主开展本单位的治安保卫工作实现的。
        
    3. （3）动员和指导学校参与治安工作：公安机关与警察在动员、指导学校参与社会治安工作时，主要是通过参与、督促、指导学校教育青少年、保护未成年人，并对其不良行为及时进行预防和矫治来实现的。

    1. |组织|简介|

### 服务群众

1. **1、服务群众的要求**：
    
    1. （1）有热心：在面对办事群众时做到“来有迎声，去有送语” ，主动关心群众生活，倾听群众诉求。例如：社区民警主动帮助联系、解决群众求助事宜。
        
    2. （2）有爱心：时刻关心群众疾苦， 了解群众需求，在法律允许的范围内尽最大努力满足群众的要求。例如：社区民警对孤寡老人、残疾人等有特殊困难的群众实行上门服务。
        
    3. （3）有耐心：面对群众要不急躁、不厌烦。例如：对于对社区民警入户走访有“敌意”的群众进行耐心沟通，笑脸相迎，从而拉近警民关系。
        
    4. （4）有恒心：服务群众是持久性工作，若没有恒心和毅力，不但会严重影响公安队伍形象，更是让群众产生做事没谱的感觉，影响服务群众的效果。
        
2. **2、服务群众的方法**：
    
    1. （1）便民利民服务：放管服，就是简政放权、放管结合、优化服务的简称。“放”即简政放权，降低准入门槛。“管”即公正监管，促进公平竞争。“服”即高效服务，营造便利环境。。比如：异地办理补换居民身份证、自助办理开具临时身份证明、网上办理机动车驾驶证、办理出入境证件“只跑一次”等。
        
    2. （2）帮扶弱势群体、上门为群众提供服务：对于存在生活困难的群众，特别是行动不便的孤寡老人 ，社区民警应该做到主动上门服务，不仅仅在办理户口、身份证、暂住证等方面给予帮助，也应该在生活上为他们排忧解难。对于辖区内的思想动荡人员进行多次拜访，了解并关心其生活。
        
    3. （3）开展咨询服务：对于群众提出的咨询、求助要求，要及时做出解答，能够办理的要及时办理，不能办理的要给出自己的建议和意见。可以通过设立接待群众日，来开展咨询工作。社区民警的咨询服务还可以利用信息化手段进行。例如，建立网上警务室、网上咨询网站，通过开设论坛、专门博客，建立 QQ 群等，在网上积极宣传和回答群众的问题，建立起为群众服务的新渠道。
        
    4. （4）及时调处矛盾纠纷：对群众反映的矛盾纠纷，有责任也有义务将这些矛盾纠纷化解于无形。要详细查清原因，形成笔录，在积极发现矛盾纠纷的同时，最大限度地避免矛盾激化 ，根据事实和有关法律规定 ，分清当事人应承担的责任 ，反复工作 ，以法正人、以理服人、以情感人 ，做好矛盾双方当事人的疏导工作 ，及时签订调处责任书。调解结束后，要对双方思想出现反复的进行回访，维护社会的和谐稳定。

## 二、行政管理能力

### 调查研究

1. **1、概念**：调查研究是公安派出所社区民警的工作方式之一，是社区民警采用走进社区、村庄，深入居民家庭及辖区机关、学校、企事业单位的方式，详细登记、核对、了解实有人口、行业场所、出租房屋等情况，对发案单位、居民进行回访，了解最新线索，征求意见和建议的警务活动。
    
2. **2、任务**：采集、了解、核对人口信息；落实高危人群的调查和管控措施；了解、掌握社情民意；回访发案的居民家庭；宣传安全防范知识与技能；开展矛盾纠纷排查；服务群众，扶危帮困。
    
3. **3、要求**：学习思考、自我提高；上门入户、沟通交流；访贫问苦、帮危济困；宣传发动、组织实施；帮教管控、引导转化；排查矛盾、化解纷争；观察搜集、研判分析；记录积累、总结归纳。
    
4. **4、实施**：调查走访前的准备(确定走访对象、确定走访时机、确定携带物品)；实施过程(接近被访者、有效进行提问、有效听取回答、适时结束走访)。
    
5. **5、内容**:
    
    1. （1）影响治安秩序的因素调查：包括政治因素、经济因素、文化因素。
    2. （2）社会治安状况调查：
        1. ①公安机关机构设置、警力状况、队伍素质、装备水平、运行状况工作效率等。
        2. ②群众性治安保卫力量及活动情况。
        3. ③保障社区治安秩序稳定的控制力状况。
        4. ④违法犯罪动向。
        5. ⑤社会热点问题及群体性事件情况。
        6. ⑥治安灾害事故发生及造成损失的数量及情况。
        7. ⑦群众对治安秩序的总体评价。
        8. ⑧群众对违法犯罪的态度。
        9. ⑨群众对公安机关、司法部门的信任感以及工作效率和作风的评价等。
        10. ⑩互联网上的不良信息或不良言论，极易对公众产生消极和负面影响，形成影响治安秩序的因素。
6. **六、公安管理业务调查**：
    
    1. （1）户口调查：户口调查是指通过人口调查来了解治安秩序情况。其调查的主要内容包括常住人口情况、暂住人口情况及重点人口情况。
        1. ①常住人口的情况。主要包括常住人口的身份、经济状况、现实表现、经常往来的社会关系及流向动态情况。
        2. ②暂住人口的情况。主要包括暂住人口的构成、来源、流向、暂住情况。时间、地点、原因、活动或就业情况、收入情况、生活状况等。
        3. ③重点人口的情况。主要包括其姓名、性别、年龄、文化程度、职业或特殊职业的人员、体貌特征、行为习惯、语言特征、婚姻状况、工作单位、家庭成员情况以及前科劣迹、主要社会关系和经常交往的人员、经济状况、现实表现等。
    2. （2）公共复杂场所、特种行业等场所情况调查：公共复杂场所指的是人员拥挤、情况复杂，容易被为违法犯罪分子所利用实施违法犯罪行为的公共场所。特种行业指在工商服务业中，因经营业务的内容和性质而易于被犯罪分子所利用，由国家或地方性法规规定交由公安机关实行治安行政管理的行业。调查的内容如下
        1. ①是否具有合法经营资格
        2. ②是否具备相应的安全条件
        3. ③是否建立并落实了各项风险防范制度
        4. ④是否配备有相应的治安防范力量及治安防范设施
        5. ⑤是否存在安全隐患
        6. ⑥是否存在违规违法经营问题及场所内是否发生过违法犯罪活动等方面的情况
7. **七、调查研究的方法**：
    
    1. （1）观察法：利用人的感觉器官来获取信息的方法。
    2. （2）访谈法：民警通过与被调查者面对面的交谈来获取信息的方法。
    3. （3）问卷法：通过设计调查问卷，由被调查者来回答的方式达到调查的目的。
    4. （4）查阅法：通过查阅资料、档案、来信、来函等各种文字、图形等来收集社会治安信息的方法。
    5. （5）技术手段调查法：技术手段调查法是指公安机关的工作人员依法运用录音录像，电视监控等手段了解和捕捉相关的信息。

### 纠纷化解

1. **1、矛盾纠纷概念**：矛盾纠纷，是指发生在居民之间、居民与法人之间和其他社会组织之间涉及人身权利、财产权利等权利义务争议的各种矛盾纠纷和日常生活中的争执，它是可能引发重大治安问题、刑事案件和群体性事件的隐患、冲突和苗头。
    
2. **2、矛盾纠纷的主要类型表现**：
    
    1. （1）损害赔偿纠纷。主要包括因殴打他人、故意伤害而产生的伤害赔偿纠纷、毁坏财物赔偿纠纷等。
    2. （2）债权债务纠纷。常见的有民间借贷纠纷、买卖合同纠纷、劳资纠纷、租赁纠纷等。
    3. （3）邻里物业纠纷。包括乱丢乱放垃圾、饲养宠物、噪音污染、违章搭建、物业管理纠纷等。
    4. （4）婚姻家庭纠纷。主要包括离婚财产分割及子女抚养、老人赡养、家庭暴力等。
3. **3、矛盾纠纷排查的重点**：
    
    1.   矛盾纠纷排查，是指民警通过走访调查、接警处警、情报搜集、综合研判等方法，发现和掌握矛盾纠纷的活动。
    2.   排查重点包括特殊利益群体可能对党政机关造成冲击的矛盾纠纷 ，涉及民族宗教等敏感问题的矛盾纠纷 ，可能引发涉外事件的矛盾纠纷 ，发生在敏感时期、敏感地区可能引发连锁反应的矛盾纠纷等。
4. **4、矛盾纠纷的特点**：
    
    1. （1）酝酿时间的长期性。大多数矛盾纠纷往往是长年累月的鸡毛蒜皮小事达到临界点继而厚积薄发的结果。
    2. （2）纠纷性质的复杂性。大多数矛盾纠纷是公说公有理，婆说婆有理，在介入初期其是非曲直较为模糊，不易为民警轻易判断，必须经过深入细致的调查了解才能初步断定理在何方。
    3. （3）参与人员的群体性。大多数矛盾纠纷初期表现为个体纠纷，一旦处理不及时，就会有家庭成员、家族人员甚至社会人员的不断介入，从而扩大事态，升级矛盾。
    4. （4）表现形式的激烈性。相当一部分矛盾纠纷初期表现形式较为平和，但随着时间的积累，就有从先期的唇枪舌剑演变为兵戎相见，矛盾日益白热化，从而不易于控制和调处。
    5. （5）诱发因素的类似性。绝大多数矛盾纠纷的诱发因素不外乎经济利益和面子问题作祟，当事人要么为了争得自己认为理所应当取得的经济利益，要么是自尊心受到刺激，从而拼死一搏，想取得自己理想的结果。
    6. （6）调处工作的艰巨性。大多数矛盾纠纷由于是在长期磕磕绊绊中“成长壮大”的，随着参与人员的日益增多、表现形式的日益激烈，加之利益因素或面子问题作怪，调处工作十分艰巨。
5. **5、矛盾纠纷排查的基本方法**：
    
    1. （1）日常排查：社区民警通过走访、调查、清查、接警出警等日常管理防范工作，随时了解、发现和掌握各种矛盾纠纷的经常性活动。
    2. （2）定期排查：派出所每月对辖区存在的矛盾纠纷进行一次全面排查和梳理，厘清矛盾焦点，抓住重点问题，制定和落实应对措施，汇总上报的活动。
    3. （3）专项排查：在重大节日、重大活动和敏感时期来临前，结合辖区实际，针对特定问题、特定对象进行的预防性排查活动。
    4. （4）联合排查：在地方党委政府和公安机关统一领导下，联合有关单位，针对多发、复杂、突出、反复出现的矛盾纠纷进行的联合排查、综合治理的活动。
6. **6、现场调解纠纷方法**：在调解矛盾纠纷的整个过程中对主要环节的把握，应参照治安调解的基本程序，采用“五步法”。
    
    1. （1）第一步：控制现场，了解相关情况；
    2. （2）第二步：耐心倾听，找准矛盾纠纷的焦点；
    3. （3）第三步：征询双方的意见，寻找共同点；
    4. （4）第四步：明法析理做工作，找准切入点；
    5. （5）第五步：寻找结合点，作出调解结论。
7. **7、典型矛盾纠纷现场处置方法**：
    
    1. **（1）物业管理纠纷**
    2.   1.制止双方过激行为，纠纷中有违法犯罪行为的，立即受案进行调查；
    3.   2.询问纠纷双方关系，判明是物业公司、开发商、物业自治管理组织等与业主纠纷，还是业主间纠纷；
    4.   3.进一步了解纠纷类型是物业管理公司侵犯业主权益的纠纷，还是开发商遗留问题引起的纠纷 ，对属于公摊面积不清 ，房屋质量不合格等房屋问题的，可以通知房屋建设部门到场处置；
    5.   4.对于合同纠纷、侵权纠纷的，可以进行必要的现场调解，调解不成的，明确告知争议双方公安机关无权处理，应通过司法及行政救济渠道解决。
    6. **（2）医患纠纷**
    7.   1.制止双方过激行为，纠纷中有违法犯罪行为的，立即受案进行调查；
    8.   2.了解纠纷基本情况，判明是基于医疗过错争议产生的医疗纠纷还是与医疗过错无关的其他医患纠纷；
    9.   3.劝离围观群众，恢复医疗机构正常工作秩序，要求医疗机构内保人员，维护现场秩序，开展劝解工作；
    10.   4.及时通知院方派员进行现场处置，也可告知患方可以向院内主管部门及院领导反映问题；
    11.   5.以不涉及争议的实证性问题为原则，进行必要的现场调解；
    12.   6.对于可能有医疗过错的，告知患方可依法提出认定。
    13. **（3）家庭成员、邻里间纠纷**
    14.   1.维护现场秩序，劝离围观群众，制止过激行为；
    15.   2.稳定当事人情绪，了解事情原委，依法开展现场调解，促成双方自行和解；
    16.   3.通过双方当事人所在村 (居) 委会出面调解；
    17.   4.因纠纷引发打架斗殴等违法行为的，依法制止违法行为，控制嫌疑人，收集并固定证据，开展调查处理工作。

### 风险识别

1. **1、治安风险识别定义**：
    
    1.   治安风险识别是指民警能够准确判明各类人、地、物、案件中所蕴含的风险并进行分级，以利于作出妥善的应对。治安风险识别是社会治安预警的基础和起点，如果不能准确辨识社会治安面临的风险，确定风险的性质，分析可能发生的损失，社会治安预警就可能失去预控的最佳时机，从而导致预警的失败。风险识别就是以公安机关为主要力量，以社会各界的力量为辅助，运用各种方法去辨别风险的征兆，分析出社会所面临的可能危害国家安全和社会治安秩序，危害公民生命财产安全的可能性和潜在原因，以便及时采取应对措施以防止各类治安问题的发生。
2. **2、治安风险谁来识别？**
    
    1.   治安警察、交通警察、武装警察等为主 ，也包括参与治安管理工作的辅警、治安志愿者、治安联防队员等。但辅警、治安志愿者和治安联防队员只能进行一般的询问和调解 ，没有执法权 ，而且需在人民警察的指导下开展工作。
3. **3、治安风险识别什么？**
    
    1.   可疑的人员、物品或车辆，潜在的危险和违法犯罪行为，虚假信息等；另外 ，醉酒者、精神病患者、被害客体、处于困难中的人民群众也是治安识别的重要对象。

4. D.对迫切需要处置的紧急报警，迅速前往现场开展处置工作

5. **4、社会治安风险主要包括**：
    
    1. （1）典型治安案件风险，如抢夺、诈骗、盗窃、赌博等。
    2. （2）重大刑事案件风险，主要是指造成严重社会影响的刑事案件，如连环杀人案件。
    3. （3）自然灾害风险，如破坏性地震、台风等可能造成社会秩序的混乱。
    4. （4）各类事故风险，包括道路交通事故、火灾、爆炸等。
    5. （5）群体性事件风险，如群体性上访、非法集会、抗议性游行、打砸抢烧事件等。
    6. （6）公共卫生风险，如空气污染、水污染、危险化学品泄漏或丢失、不明原因的疾病流行等。
    7. （7）恐怖袭击风险。
    8. （8）社会心理风险，如由于突发事件、错误信息等造成的社会心理恐慌等。
6. **5、治安识别的类型**：
    
    1. **（1）人的风险识别**：
        
    2.   ①证件疑点：证件疑点即指本人持假身份证，或一个人持有多个身份证或工作证等，以及本人与所持身份证上信息有明显差别或不相符，包括相貌、年龄、籍贯、口音等。其中，假身份证的疑点又包括:第一，机读疑点:第二，视读疑点，第三，逻辑疑点。
        
    3.   ②行为疑点：行为疑点是指行为举止违反正常人的行为模式，且又符合或相似于一些违法犯罪活动的特征。行为疑点包括行为本身疑点和行为发生的时间和地点疑点两个方面。行为本身疑点是指行为人的行为表现与行为应当的、正常的表现的差异。行为实施的时间和空间疑点是指行为发生的时间、地点，违背正常人的通常习惯。
        
    4.   ③神态疑点：神态疑点是指行为人表现出来的与所处的特定时间、空间环境不相适应的各种神态、表情。
        
    5.   ④痕迹疑点：痕迹疑点是指行为人人身或者其携带的物品、骑乘的车辆等具有非正常的痕迹特征。
        
    6.   ⑤物品疑点：物品疑点是指行为人携带的物品与携带者身份、目的、处所等有诸多方面的不相称，或与其自述有矛盾的物品。
        
    7.   ⑥装束疑点：装束疑点是指行为人的衣着打扮与自己的性别、年龄、职业以及体态、身高还有季节等不相符的情况。
        
    8.   ⑦体貌疑点：体貌特征方面的疑点是指行为人的相貌、体态、口音等特征与公安机关通报的违法犯罪分子相近。
        
    9.   ⑧关系疑点：关系疑点是指当有两个以上的人在一起行动时所表现出来的相互之间的不正常的关系。
        
    10. **（2）公共娱乐场所的风险识别**：娱乐场所，是指以营利为目的，并向公众开放、消费者自娱自乐的歌舞、游艺等场所。它具有公共性、盈利性、自娱自乐性等特征。
        
    11.   ①开业风险：娱乐场所不得设在下列地点：居民楼、博物馆、图书馆和被核定为文物保护单位的建筑物内；居民住宅区和学校、医院、机关周围；车站、机场等人群密集的场所；建筑物地下一层以下；与危险化学品仓库毗连的区域。
        
    12.   ②人员风险：娱乐场所应当与经公安机关批准设立的保安服务企业签订服务合同，配备已经取得资格证书的专业保安人员，并通报娱乐场所所在辖区公安派出所。并且从业人员名单应当包括从业人员的真实姓名居民身份证复印件、外国人就业许可证复印件等内容。
        
    13.   ③设施风险：娱乐场所应在疏散通道和安全出口设置明显指示标志:制订安全工作方案和应急疏散预案:保障疏散通道和安全出口畅通。在大厅、包厢、包间内显著位置悬挂禁毒、禁止卖淫娼等内容的警示标志，未成年人禁止或限入标志以及举报电话。出入口、主要通道安装闭路电视监控设备，并保证在营业期间正常运行，不得中断，并将闭路电视监控录像资料保存 30日备查。包厢、包间应安装展现室内整体环境的透明门窗，在包、包间内不得设置隔断，不得有内锁装置。营业时间内，歌舞娱乐场所包厢、包间门窗透明部分不得遮挡。娱乐场所应建立营业日志，应当留存 60 日备查。
        
    14. **（3）重点地区、单位的风险识别**：治安保卫重点单位是指关系全国或者所在地区国计民生、国家安全和公共安全的单位。治安保卫重点单位由县级以上地方各级人民政府公安机关按照下列范围提出，报本级人民政府确定：
        
    15.   ①广播电台、电视台、通讯社等重要新闻单位；
        
    16.   ②机场、港口、大型车站等重要交通枢纽；
        
    17.   ③国防科技工业重要产品的研制、生产单位；
        
    18.   ④电信、邮政、金融单位；
        
    19.   ⑤大型能源动力设施、水利设施和城市水、电、燃气、热力供应设施；
        
    20.   ⑥大型物资储备单位和大型商贸中心；
        
    21.   ⑦教育、科研、医疗单位和大型文化、体育场所；
        
    22.   ⑧博物馆、档案馆和重点文物保护单位；
        
    23.   ⑨研制、生产、销售、储存危险物品或者实验、保藏传染性菌种、毒种的单位；
        
    24.   ⑩国家重点建设工程单位；
        
    25.   ⑪其他需要列为治安保卫重点的单位。
        
    26. **（4）旅馆业的风险识别**：经营接待旅客住宿的旅馆、饭店、宾馆、招待所、客货栈、车马店、浴池等属于旅馆业的范围。随着社会的发展，我国已将培训中心、度假村、公寓式酒店、农家乐、民宿以及提供住宿服务的洗浴、足疗、按摩等经营场所纳入旅馆业。
        
    27.   ①人员风险：从业人员是否合格。
        
    28.   ②场地风险：营业场所建筑设施和设备是否符合安全标准。
        
    29.   ③制度风险：开业审批和登记备案制度；登记验证制度；旅客财物保管制度；值班与来访人员管理制度；情况报告与协查制度；从业人员培训制度。
        
    30. **（5）危险物品从业单位的风险识别**：危险物品是指具有杀伤、爆炸、易燃、毒害、腐蚀、放射性等性质，在生产、储存、运输、销售、使用和销毁过程中，容易造成人身伤亡和财产损失或可能危害公共安全而需要特别防范或管制的物品。公安机关管理危险物品的范围是：枪支弹药、管制刀具、爆炸物品(包括民用爆炸物品和烟花爆竹)、剧毒物品、放射性物品。
        
    31.   ①单位和人员风险。危险物品从业单位的资质和从业人员的资质是否符合规定，人员现场值守情况等。
        
    32.   ②危险物品状态。危险物品来源和流向是否合法，物品的品种、数量是否与核定品种、数量相符，性质相抵触的危险物品是否分库存放，物品码放的位置和高度是否符合安全要求等。
        
    33.   ③设施设备。
        
    34.   ④管理制度。主要包括危险物品从业单位的各项规章制度、管理账册、安全保卫方案、突发事件应急处置预案等建立和落实情况。
        
    35. **（6）大型活动的风险识别**：大型活动是指在特定的时间、空间所进行的，由众多人员参加的，具有一定影响的有益的活动。例如文艺演出、体育比赛、展览展销、招聘会等。

## 三、信息工作能力

### 情报信息收集

1. **1、信息收集定义**：信息收集是指相关主体根据工作需要，通过一定的渠道，按照一定的程序，采用科学的方法，有组织、有计划对各种有价值的信息进行搜集的过程。
    
2. **2、公安信息收集的方法**：
    
    1. （1）一般调查方法：问卷调查，专访，咨询等。
    2. （2）特殊调查方法：审讯，勘查，侦查，监控等。
3. **3、公安业务收集的范围**：
    
    1. **（1）人员要素**
    2.   违法犯罪嫌疑人信息是公安机关关注的重点。包括违法犯罪嫌疑人的自然信息、行为信息和现实信息。
    3.   自然信息，是指犯罪嫌疑人区别于他人所必备的基本信息，是能够反映人体特征的信息。
    4.   行为信息，是指犯罪嫌疑人在实施犯罪时所必然要出现的信息。
    5.   现实信息，是指犯罪嫌疑人目前的现实表现、经济收支、接触人员、活动场所，以及其住址变迁、死亡、重新犯罪和体貌特征的变化等动态信息。
    6. **（2）团伙要素**
    7.   成员情况、机构活动、成员或组织的案件情况
    8. **（3）物品要素**
    9.   损失物品情况，如物品名称、类型、品牌、型号、产地等:
    10.   现场痕迹、作案工具和遗留在作案现场的各种物品等。作案工具包括枪支弹药、刀具、机动车等。其中，车辆要素收集:车辆信息和车主信息。
    11. **（4）车辆要素**
    12.   车辆信息、驾驶人员信息
    13. **（5）案(事)件要素收集**
    14.   案(事)件类别、危害程度以及作案时间和地点、作案手段、作案过程、逃离现场的方法等。
4. **4、社会信息收集**：
    
    1. **（1）敌情情报信息**
    2.   敌对组织、民族分裂势力、宗教极端势力、恐怖组织等有关危害国家安全的情报信息。
    3. **（2）维稳情报信息**
    4.   大规模游行、请愿、上访、罢市等集体活动、恶性突发事件和恶性刑事案件等影响社会稳定的事件和因素等情报信息。
    5. **（3）舆情情报信息**
    6.   重大会议、方针政策以及战略部署的情报信息，民众在媒介中对社会重大事件的反映等信息。

7. **2、话单分析**
    1.   话单分析法是指通过研究和解析目标对象的电话通信记录，了解目标对象的联系关系及亲疏程度、目标对象的活动范围及活动方式，从而掌握其社会关系网和活动规律的分析方法。
    2.   话单分析的内容主要包括机主资料、主叫号码、被叫号码、通话时间、通话时长、主叫归属地、被叫归属地、电子串号、基站信息。
    3. **话单分析的方法：**
    4.   （1）频率分析：是指在一定范围内，将某一话单信息重复出现的次数进行统计排序分析的方法。通过频率分析，可以发现目标对象的主要联系人、通话时段规律、主要活动地点等重要线索。
    5.   （2）时段分析：对案发前、案发中、案发后的通话差异进行分析，一般而言，犯罪嫌疑人的话单呈现如下特征，如果出现以下2个特征，则可将此话单作为主线进行跟进分析。
        1.   ①案发前通话次数平稳，接近案发时通话量激增。
        2.   ②案发前机主通话出现规律性呼叫。
        3.   ③案发时突然出现平时不联系的号码。
        4.   ④案发后通话量变少，甚至该号码从此消失。
    6.   （3）空间分析：根据话单记录的“漫游地” “呼叫发起地”以及“小区代码”等信息，对犯罪嫌疑人主叫、被叫电话的空间地点进行分析，从而得出持机人的行动轨迹。
        1.   ①可以分析犯罪嫌疑人案发时是否到过现场，案发后的逃跑路线及藏匿地点。
        2.   ②可以判断嫌疑人活动轨迹。
        3.   ③在系列案件中，可以将多个发案地点在发案时间的通信情况进行号码碰撞得出交集，并从中查找犯罪嫌疑人手机号码。

    1. （1）物品流程图
    2.   物品流程图是将案件中错综复杂的物品流动通过可视化的方式展现出来的一种技术。它可以将一个复杂案件中的物品流动清晰地展现在分析人员面前，通过对与犯罪相关的物品流动过程的认识可以加强对犯罪过程或犯罪组织的认识。在物品流程图中实体(如人、地点、组织、电话、银行账户 ) 是中心环节。代表相关实体的符号、连线和箭头，显示了物品在实体之间的流动。

    5.   一是哪些行为可以独立于其他行为。
    6.   二是哪些行为是某一行为的先行行为。
    7.   三是哪些行为是某一行为的衍生行为。

### 盘查

> 盘查，是指公安机关人民警察在执行勤务过程中，为维护公共安全，预防、发现、控制违法犯罪活动而依法采取的盘问、检查等行为。《人民警察法》第九条：为维护社会治安秩序，公安机关的人民警察对有违法犯罪嫌疑的人员，经出示相应的证件，可以当场盘问、检查。

1. **1、盘查程序**：
    
    1. （1）第一步：告知(敬礼、亮明警察身份、说明理由)
    2. （2）第二步：盘问(确定对方身份)
    3. （3）第三步：检查(人身、物品)
    4. （4）第四步：处理结果(放行或继续盘问)
2. **2、盘查要求**：民警执行盘查任务时，应始终坚持理性、平和、文明、规范，因情施策，确保安全。
    
    1. （1）民警执行盘查任务时，应当着制式服装；未着制式服装的，应当出示人民警察证；应当向被盘查人敬礼并告知：“我是 xxx (单位) 民警，现依法对你进行检查，请你配合。”盘查排除违法犯罪嫌疑后，民警应当向被盘查人敬礼，并说“谢谢你的合作”，礼貌让其离去。
    2. （2）民警应当选择光线较好、场地开阔、有依托或者容易得到支援的场地或者道路等作为盘查地点，盘查过程中应当保持高度警惕，注意被盘查人的身份、体貌、衣着、行为、携带物品等可疑之处，随时做好应对突发情况的准备。
    3. （3）盘查一般由两名以上民警进行，并明确警戒和盘查任务分工。
    4. （4）盘查可疑人员时，应当遵守的规定：
        1. ①与被盘查人保持适当距离，尽量让其背对开阔场地；
        2. ②对有一定危险性的违法犯罪嫌疑人，先将其控制并进行检查，确认无危险后方可实施盘问；
        3. ③站位合理，分工明确。
        4. ![](https://www.saduck.top/ga/jczwt.png)
            
        5. ④盘问时由一人主问，其他人员负责警戒，防止被盘查人或者其同伙的袭击。盘查多名可疑人员时，民警应当责令所有被盘查人背对开阔场地，并在实施控制后，分别进行盘查。当盘查警力不足以有效控制被盘查人时，应当维持控制状态，立即报告，请求支援。
3. **3、盘查内容**：
    
    1. （1）身份证
    2. （2）人身检查：对经过盘问，确认有违法犯罪行为或者嫌疑不能排除的，应当先对被盘查人依法进行人身检查，并进一步检查其携带物品。
    3. （3）物品检查：对可疑物品进行检查。
    4. （4）可疑车辆检查：对可疑车辆进行检查。
    5. （5）设卡检查：公安机关执行设卡检查任务对被拦截车辆进行检查时，执行盘查任务的民警应当从车辆的驾驶员一侧接近车辆，迅速控制驾驶员和车内其他人员；执行警戒任务的民警应当占据有利位置，从各个角度密切监视车内人员，车上人员应当逐一下车接受盘查。
4. **4、盘查常见的突发情况处置办法**：
    
    1. （1）以语言形式对抗盘查，民警应告知实施盘查是法律赋予公安机关人民警察的权利，必须予以配合。仍不配合的，可口头传唤，将其强行带至指定地点，在有见证人在场的情况下对其进行盘查；
    2. （2）盘查对象随意插兜或掏东西，民警应立即语言控制，责令其慢慢地将手拿出；若不听命令，应加重语言控制的力度，同时采取相应的戒备姿势，保持适当的距离；
    3. （3）盘查对象不断逼近的情况，民警应迅速拉开距离，调节站位，并做好使用相应武力控制的准备，防止出现袭警情况；
    4. （4）盘查对象突然转身离开，民警应尾随跟进，注意保持距离。追击逃跑时，尽量不要从其背后去抓，以防止对方的突然回身攻击；若目标分头逃跑时，警察必须保持警力优势追击；
    5. （5）如果盘查对象徒手袭击警察，警察应立即采用徒手或使用警械控制。在确保自身安全的前提下，迅速判明情况，作出正确评估，视情况予以处置；如果盘查对象持械袭击警察，警察应视情况采取行动，保持安全距离，并使用高一级的武力对其控制；如果盘查对象掏取枪械袭击警察，警察应迅速寻找掩体，拉开距离，拔枪准备。多名嫌疑人袭警时，应在掩体后面做好警戒，呼叫等待增援。
5. **5、继续盘问的适用条件**：
    
    1. （1）对当场盘问、检查后，不能排除其违法嫌疑，依法可以适用继续盘问的，可以将 其带至公安机关 (无需经过批准) ，经公安派出所负责人批准，对其继续盘问。
    2. （2）对违反出境入境管理的嫌疑人依法适用继续盘问的，应当经县级以上公安机关或者 出入境边防检查机关负责人批准。
6. **6、继续盘问的具体情形**：
    
    1. （1）被害人、证人控告或者指认其有犯罪行为的；
    2. （2）有正在实施违反治安管理或者犯罪行为嫌疑的；
    3. （3）有违反治安管理或者犯罪嫌疑且身份不明的；
    4. （4）携带的物品可能是违反治安管理或者犯罪的赃物的。
7. **7、继续盘问的时限**：
    
    1. （1）一般情况：不得超过 12 小时，由公安派出所负责人审批。
    2. （2）对在十二小时以内确实难以证实或者排除其违法犯罪嫌疑的：12 小时延长至 24 小时，报县、市、旗公安局或者城市公安分局的值班负责人审批。
    3. （3）对不讲真实姓名、住址、身份，且在二十四小时以内仍不能证实或者排除其违法犯罪嫌疑的：24 小时延长至 48 小时，报县、市、旗公安局或者城市公安分局的值班负责人审批。
    4. `注意：不得决定将继续盘问时限直接从十二小时延长至四十八小时。`

### 接警

> 接警是接受报警，指接警人员对报警、求助、投诉案(事)件进行询问、登记，了解情况，判明性质和轻重缓急，下达处警指令的全过程。

    1. （1）刑事案件
    2. （2）治安案件
    3. （3）危及人身、财产安全或者社会治安秩序的群体性事件
    4. （4）自然灾害、治安灾害事故

    2. （2）老人、儿童以及智障人员、精神疾病患者等人员走失，需要公安机关在一定范围内帮助查找的;
    3. （3）公众遇到危难，处于孤立无援状况，需要立即救助的
    4. （4）涉及水、电、气、热等公共设施出现险情，威胁公共安全、人身或者财产安全和工作、学习、生活秩序，需要公安机关先期紧急处置的
    5. （5）需要公安机关处理的其他紧急求助事项
3. **3、受理投诉类警情的接警范围**：
    
    1. 公安机关及其人民警察正在发生的违反《中华人民共和国人民警察法》《公安机关人民警察纪律条令》等法律、法规和人民警察纪律规定，违法行使职权，不履行法定职责，不遵守各项执法、服务、组织、管理制度和违背职业道德。

        4.   Ⅳ.本条第 3 款有特别规定的，依照执行。
    11. **（3）分类处置**：
        
    12.   ①处置案 (事) 件
        
        1.   Ⅰ.受理案 (事) 件报警后，应当根据警情立即调派警力进行处置。
        2.   Ⅱ.对危及公共安全、人身或者财产安全的紧急、重大案 (事) 件，应当在派警处置的同时，根据警情的性质、事态规模、紧急程度，立即报告分管负责人。
        3.   Ⅲ.需要进行布控查缉的，应当通知治安卡口、检查站 (查报站) 、路面交巡警等进行布控查缉。
        4.   Ⅳ.对跨区域的重大案 (事) 件，在指挥本地警力处置的同时，可以视情报告公安机关负责人，将情况报告上级公安机关或者通报有关地区公安机关。有关地区公安机关在接到上级公安机关指令或者案发地公安机关的通报后，应当迅速落实有关查缉措施，提供必要的协助，并随时与案 (事)发地公安机关或者本地公安机关 110 报警服务台 (含指挥中心) 保持联系。
    13.   ②处置群体性事件
        
        1.   Ⅰ.对规模较小、影响不大的一般性群体性事件 ，应当尽快将情况通报业务主管部门，酌情派警维持现场秩序，协助有关部门进行疏导劝阻，防止事态扩大。
        2.   Ⅱ.对规模较大、行为方式激烈的群体性事件，应当立即报告分管负责人。
    14.   ③处置灾害、事故：应当根据灾害、事故的种类、程度，派警处置，并报告分管负责人
        
        1.   Ⅰ.发现纵火嫌疑的，应当同时指令事发地刑侦部门出警侦查。
        2.   Ⅱ.发生山林火灾时，应当先通报森林防火指挥部，根据其要求派警。
        3.   Ⅲ.对道路交通事故警情 ，应当立即向交警部门下达处警指令，并根据警情处置需要或者交警部门的要求，视情决定调派辖区公安派出所或者就近警力协助。
        4.   Ⅳ.涉及交通肇事逃逸的 ，应当及时组织实施堵截、追缉 ，并视情通报相邻公安机关布控、协查。
        5.   Ⅴ.因交通事故引发纠纷的，同时指令巡警或者事发地公安派出所派警协助处置。
        6.   Ⅵ.对涉及危险化学品的交通事故，可能产生有毒有害物质或者危险物品泄漏的，应当指令交警、治安、巡警等部门出警，并立即通报政府有关职能部门赶赴现场共同处置。
    15.   ④处置求助
        
        1.   Ⅰ.应当及时指令警力处置。
        2.   Ⅱ.对人员失踪警情，应当及时组织治安、派出所、巡警、交警等街面执勤力量查找，有被侵害可能的，通知刑侦部门侦查。
        3.   Ⅲ.对溺水等警情，应当立即指派就近民警携带救援设备到场处置，并通知有关专业救援力量到场处置。
    16.   ⑤处置投诉
        
        1.   Ⅰ.对公安机关和民警正在发生的问题投诉的，应当指令就近警力先期处置，同时通知警务督察部门进行现场调查和处理。
        2.   Ⅱ.对公安机关和民警既往发生的问题投诉的，视具体情况移交本级纪检、监察、信访、法制或者其他有管辖权的部门进行调查和处理。
        3.   Ⅲ.对已通过其他渠道投诉的，应当通知或者交由原受理部门处理。
        4.   Ⅳ.投诉辖区内的外地公安民警或者其他无隶属关系的公安民警的，应当指令就近警力先期处置，同时通知本级警务督察部门前往调查和处理，并由警务督察部门移送被投诉人所属单位处理。
    17.   ⑥处置涉民族宗教警情：对涉及民族宗教事务的警情，除指令有关处警单位和人员处置外，应当及时通知政府民族宗教事务部门派人协助开展处置工作。
        
    18.   ⑦处置涉外警情：对涉及外籍人员的警情，除指令有关处警单位和人员处置外，应当及时通知公安机关出入境管理部门派人协助开展处置工作。
        
    19.   ⑧增派警力：到达现场的处警民警不足以制止或者控制现场局面，请求增援的，应当按照工作预案，迅速调集、指挥有关警种、部门赶赴现场增援。
        
    20. **（4）信息沟通**：
        
    21.   ①在案 (事) 件处置过程中 ，110 报警服务台 (含指挥中心) 应当了解、收集处警民警到达现场的先期报告和案 (事) 件处置完毕后的反馈报告。对重大案 (事) 件或者性质一时难以判明的案 (事)件，进行跟进了解，随时掌握案情进展和处置工作情况。对收集到的相关信息和反馈情况，认真做好记录。
        
    22.   ②110 报警服务台 (含指挥中心) 对当天接报、掌握的重要警情信息，应当严格依照重要情况信息收集以及报送工作的规定办理，并定期对警情进行研判，发布预警通报。
        
    23. **（5）回复、回访**：110 报警服务台应当依照有关规定或者在必要时将处警结果回复报警人、求助人、投诉人，并回访报警人、求助人、投诉人。
        
    24. **（6）保密**：报警人、求助人、投诉人如果不愿公开自己的姓名和报警、求助、投诉的行为，应当为其保守秘密，并在接处警记录中注明。对投诉内容及投诉人情况，严禁泄露给被投诉对象或者其他无关人员。

### 处警

> ‌处警‌指公安部门在接到指令或报警后，前往现场处置紧急或危险情况，并对报警人及相关人员进行调查处理的完整过程。‌

1. **1、及时到达现场**：
    
    1. （1）“与案件性质、规模相当”最低不得少于两名正式民警。见习民警、辅警等不具备执法资格的人员不得单独处警。
    2. （2）按规定着装警容严整携带必要的警械、通讯工具等处警装备。
    3. （3）开启录音、录像设备进行录音、录像、拍照，及时收集现场证据。
    4. （4）立即向 110 指挥中心报告现场情况。
    5. （5）出示人民警察证，表明警察身份(着警服可不出示，对方要求出示则必须出示)。
    6. （6）快速判断警情，采取相应处置措施。
2. **2、制止违法犯罪**：
    
    1. **（1）处置措施的种类**：公安民警为现场制止违法犯罪行为而采取的强制手段，由轻到重依次为：口头制止、徒手制止、使用警械制止、使用武器制止。
        
    2. **（2）现场处置实施程序**
        
        1. ①处置措施前，表明身份+出示证件。着警服可不出示，对方要求出示则必须出示。
        2. ②现场制止，应当按照《 110 接处警工作规则》做好处警记录。
        3. ③到达处置现场后，应当与所属公安机关保持联络，迅速报告。必要时，请求增援。
        4. ④现场制止违法犯罪行为时，应当根据现场警情的性质、危害程度、影响范围、涉及人数、当事人身份及警情敏感性等综合因素，快速判断，采取相应的处置措施。
        5. ⑤使用较重处置措施时，可以同时使用较轻处置措施作为辅助手段。
        6. ⑥违法犯罪行为后，对可能脱逃、行凶、自杀、自伤或者有其他危险行为的违法犯罪行为人，可以使用手铐、警绳等约束性警械将其约束，并及时收缴其所持凶器。
    3. **（3）使用处置措施的基本要求**：适度、安全
        
    4. **（4）口头制止**
        
        1. ①使用条件：对正在以非暴力方式或者以轻微动作实施违法犯罪行为的；注意：口头制止若致嫌疑人可能逃窜、毁证或酿更重后果，可依规徒手制止。
        2. ②口头制止内容：明确指令违法行为人停止行为、表明警察身份与执法依据、警示法律后果、引导现场秩序。
        3. ③用语要求：明确、简洁、易懂，禁止使用侮辱性、歧视性语言。
        4. ④调整措施：违法犯罪行为人不听从口头制止的，可以将其传唤至公安机关处理。
    5. **（5）徒手制止**
        
        1. ①使用条件：对正在以轻微暴力方式实施违法犯罪行为，尚未严重危及公民或者民警人身安全，经警告无效的，以及情况紧急，来不及警告或者警告后可能导致更为严重危害后果的。
        2. ②使用限度：徒手制止的，应当以违法犯罪行为人停止实施违法犯罪行为为限度；除非必要，应当避免直接击打违法犯罪行为人的头部、裆部等要害部位。当违法犯罪行为人停止实施违法犯罪行为时，公安民警应当立即停止可能造成人身伤害的徒手制止动作，并依法使用手铐、警绳等约束性警械将其约束。
    6. **（6）使用警械制止**
        
        1. ①使用条件：公安民警遇有《中华人民共和国人民警察使用警械和武器条例》第七条所列危害公共安全、社会秩序、公民人身安全等情形之一，经警告无效的，可以使用警棍、催泪喷射器等驱逐性、制服性警械。
        2. > 《中华人民共和国人民警察使用警械和武器条例》第七条  
            > 人民警察遇有下列情形之一，经警告无效的，可以使用警棍、催泪弹、高压水枪、特种防暴枪等驱逐性、制服性警械：  
            > 1.结伙斗殴、殴打他人、寻衅滋事、侮辱妇女或者进行其他流氓活动的；  
            > 2.聚众扰乱车站、码头、民用航空站、运动场等公共场所秩序的；  
            > 3.非法举行集会、游行、示威的；  
            > 4.强行冲越人民警察为履行职责设置的警戒线的；  
            > 5.以暴力方法抗拒或者阻碍人民警察依法履行职责的；  
            > 6.袭击人民警察的；  
            > 7.危害公共安全、社会秩序和公民人身安全的其他行为，需要当场制止的；  
            > 8.法律、行政法规规定可以使用警械的其他情形。
            
        3. ②操作规程：使用催泪警械（根据现场情况，要求现场无关人员躲避；选择上风向站位和安全有效距离，直接向违法犯罪行为人喷射催泪剂；制服以后，对有异常反应或者引发疾病的，应当及时采取适当措施救治）。公安民警制服违法犯罪行为人后 ，应当立即使用手铐、警绳等约束性警械将其约束。对受伤的违法犯罪行为人，应当及时采取适当措施救治。
    7. **（7）使用武器制止**
        
        1. ①使用条件：公安民警遇有《中华人民共和国人民警察使用警械和武器条例》第九条规定的紧急情形之一，经警告无效的，可以使用武器；来不及警告或者警告后可能导致更为严重危害后果的，可以直接使用武器。使用枪支，包括持枪戒备、出枪警示、鸣枪警告、开枪射击行为。
        2. > 《中华人民共和国人民警察使用警械和武器条例》第九条  
            > 人民警察遇有下列情形之一，经警告无效的，可以使用武器：  
            > 1.放火、决水、爆炸等严重危害公共安全的；  
            > 2.劫持航空器、船舰、火车、机动车或驾驶车船等交通工具故意危害公共安全的；  
            > 3.抢夺、抢劫枪支弹药、爆炸、剧毒等危险物品，严重危害公共安全的；  
            > 4.使用枪支、爆炸、剧毒等危险物品实施犯罪或以暴力抗拒、阻碍人民警察依法履行职责的；  
            > 5.暴力袭击人民警察，危及人民警察生命安全的；  
            > 6.劫持人质，危及公民生命安全的；  
            > 7.实施凶杀、劫持人质等暴力行为，危及公民生命安全的；  
            > 8.国家规定的警卫、守卫、警戒对象和目标受到暴力袭击、破坏，或者有受到暴力袭击、破坏紧迫危险的；  
            > 9.聚众械斗、暴乱等严重破坏社会治安秩序，用其他方法不能制止的；  
            > 10.法律、行政法规规定可以使用武器的其他情形。
            
        3. ②操作规程：判明现场情况；表明警察身份，出枪示警；情况紧急时，可以在出枪的同时表明身份；命令在场无关人员躲避；命令犯罪行为人停止实施暴力犯罪行为，或者鸣枪警告；犯罪行为人在公安民警口头警告或者鸣枪警告后继续实施暴力行为的，可以对其使用武器；来不及警告或者警告后可能导致更为严重危害后果的，可以直接使用武器；犯罪行为人停止实施犯罪，服从公安民警命令，或者失去继续实施犯罪能力的，应当立即停止射击，并持枪戒备；在未确定危险消除前，应当继续保持持枪戒备；确认危险消除后，应当关闭枪支保险，收回枪支。
3. **3、控制现场**：
    
    1. **（1）控制事态**
        
        1. ①处警民警判明警情后，应当依照前条规定采取有效措施制止违法犯罪行为，制服、约束违法犯罪行为人，控制危险物品。
        2. ②必要时，依法采取交通管制措施，疏散周边或者围观人员。
        3. ③对严重危害社会秩序的突发事件，可以依法实行现场管制。
    2. **（2）设置警戒**：必要时，设置警戒区域，使用警戒带或警示灯进行警戒，严禁无关人员进入警戒区域。
        
    3. **（3）排除险情**
        
        1. ①抢救人员。在做好自身安全防护的情况下，立即抢救遇险人员，疏散周边人员至安全地带，做好外围警戒。
        2. ②专业排险。对爆炸性、腐蚀性、放射性、毒害性、传染性等专业性较强的排险工作，应当报告下达处警指令的单位联系专业人员到场处置。
    4. **（4）保护现场**：在现场设置警戒区域，无关人员不得擅自进入警戒区域，不得触动现场上的痕迹、物品和尸体，并将现场保护情况及时报告现场勘验、检查指挥人员。
        
    5. **（5）应对采访拍摄**
        
        1. ①未经授权，任何单位和民警不得接受媒体记者采访或者公开发布有关案(事) 件信息，现场处警民警应当以正在执行公务为由婉拒记者、群众的询问。
        2. ②现场有媒体记者采访或者群众拍摄执法活动的，应当请其在警戒区外采访、 拍摄。对设立专门采访区域的，可以告知媒体记者到指定区域，有专人负责；对未设立 专门采访区域的，可以告知媒体记者向公安机关新闻宣传部门提出采访申请。
        3. ③对上级或者本级公安机关同意采访的，应当配合做好采访工作。
4. **4、救治伤员**：
    
    1. **（1）处警民警的处置程序 (现场有人员伤亡的)**
        1. ①立即向下达处警指令的单位口头报告。报告内容包括民警身份及联系方式、 目前所处位置、现场情况、人员伤亡情况、采取处置措施情况；异地采取处置措施的， 应当同时向现场所在地公安机关口头报告上述情况。
        2. ②立即救助伤员。根据伤势危急程度和所掌握的救护知识，迅速对受伤人员采 取临时救治措施。根据需要立即通知 120 急救中心到场救治 ，或者报告下达处警指令 的单位，由其通知 120 急救中心到场救治。
    2. **（2）现场所在地县级以上公安机关的处置程序**
        1. ①视情指派有关部门立即赶赴现场，进行现场勘查等工作。对杀人、放火、爆炸等严重犯罪的现场或者已经造成人群聚集的现场，公安机关负责人应当赶赴现场进行指挥。
        2. ②必要时，设置警戒。
        3. ③及时对受伤人员采取紧急救治措施。尚未通知 120 急救中心派员到场的，视情通知 120 急救中心派员到场救治。
        4. ④因采取处置措施造成人员重伤、死亡的，及时通知当地人民检察院。
        5. ⑤将伤亡人员情况及时通知其家属或者其所在单位，并做好善后工作。
        6. ⑥根据需要准备新闻口径，依照有关规定做好舆论宣传、引导工作。
5. **5、现场追捕**
    
    1.   处警民警到达现场发现违法嫌疑人逃离现场的，应根据现场获取的信息实施搜索、堵截、追捕，或者根据具体情况报告上级或 110 指挥中心，迅速布置警力在主要路段，关卡实施堵截、检查。
6. **6、调查取证**
    
    1. 需要调查取证的，应当针对现场情况和警情特点，及时、全面收集、固定证据，查明当事人基本情况，案 (事) 件的原因、过程和结果，行为手段和方式等。
7. **7、现场处结**
    
    1. **（1）结束案 (事) 件现场处置**
        1. ①当场处罚：对符合当场处罚条件的行政案件，处警民警可以当场作出处罚决定。
        2. ②现场调解：对符合现场调解条件的治安案件，处警民警可以当场进行调解。
        3. ③当场移交：对案 (事) 件主管部门、单位人员到达现场的，应当现场移交，办理 交接手续，并协助现场处理。
        4. ④继续办理：对案 (事) 件现场处理完毕，需要继续办理的，将有关人员和证据带 回公安机关继续办理。
    2. **（2）结束求助现场处置**
        1. ①救助成功的，确认被救人员身份，必要时通知其家属到场办理交接手续。
        2. ②救助不成功，须离开现场的，应当报经本单位负责人批准。
        3. ③救助结束后 ，有民事纠纷的 ，告知当事人向相关调解组织请求调解 ，或者向人民法院提起民事诉讼。
    3. **（3）结束投诉现场处置**
        1. ①对投诉现场处置完毕 ，应当将有关人员和证据材料带回 ，并转交警务督察部门或者具体承办投诉的部门、单位。
        2. ②警务督察或者具体承办投诉的部门、单位人员到达现场的，应当当场移交，办理交接手续，并协助现场处理。
8. **8、后续办理**
    
    1. （1）呈报《接处警登记表》。
    2. （2）紧急实施行政强制措施报告。
    3. （3）伤亡报告。
    4. （4）使用武器报告。

## 五、应急处理能力

### 突发事件

1. **1、突发事件的概念**：突发事件，是指突然发生，造成或者可能造成严重社会危害，需要采取应急处置措施予以应对的自然灾害、事故灾难、公共卫生事件和社会安全事件。
    
2. **2、突发事件的分类**：
    
    1. |突发事件分类|突发事件常见种类|
        |---|---|
        |自然灾害|水旱灾害，气象灾害，地震灾害，地质灾害，海洋灾害，生物灾害和森林草原火灾|
        |事故灾难|工矿商贸等企业的各类安全事故，交通运输事故，公共设施和设备事故，环境污染和生态破坏事件等|
        |公共卫生|传染病疫情，群体性不明原因疾病，食品安全和职业危害，动物疫情，以及其他严重影响公众健康和生命安全的事件|
        |社会安全|恐怖袭击事件，经济安全事件、群体性事件和涉外突发事件等|
        
3. **3、领导职责**：
    
    1. |突发事件应对中的领导职责|群体性事件的现场处置工作领导职责|
        |---|---|
        |①由县级人民政府负责本行政区域内的突发事件处置，并立即向上一级人民政府报告，必要时可越级报告；  <br>②涉及两个以上行政区域的，由有关行政区域共同的上一级人民政府负责，或由各有关行政区域的上一级人民政府共同负责|①由事发地的县级公安机关负责，上级公安机关可以派人到场指导或直接负责处置工作；  <br>②跨地区发生的，由共同的上一级公安机关负责也可以由上级公安机关指定的公安机关负责|
        
4. **4、预警制度**：
    
    1. （1）分级的依据：紧急程度、发展势态和可能造成的危害程度
    2. （2）所分级别：一级 (Ⅰ)、二级 (Ⅱ)、三级 (Ⅲ)、四级 (Ⅳ)，分别用红、橙、黄、蓝表示。
5. **5、突发事件的分级预警**：按照突发事件发生的紧急程度、发展势态和可能造成的危害程度分为一级、二级、三级和四级，一级为最高级别。
    
    1. **（1）三级、四级预警应当采取的措施**
        1. ①启动应急预案。
        2. ②责令有关部门、专业机构、监测网点和负有特定职责的人员及时收集、报告有关信息，向社会公布反映突发事件信息的渠道，加强对突发事件发生、发展情况的监测、预报和预警工作。
        3. ③组织有关部门和机构、专业技术人员、有关专家学者，随时对突发事件信息进行分析评估，预测发生突发事件可能性的大小、影响范围和强度以及可能发生的突发事件的级别。
        4. ④定时向社会发布与公众有关的突发事件预测信息和分析评估结果，并对相关信息的报道工作进行管理。
        5. ⑤及时按照有关规定向社会发布可能受到突发事件危害的警告，宣传避免、减轻危害的常识，公布咨询电话。
    2. **（2）一级、二级预警应当采取的措施**
        1. ①责令应急救援队伍、负有特定职责的人员进入待命状态，并动员后备人员做好参加应急救援和处置工作的准备。
        2. ②调集应急救援所需物资、设备、工具，准备应急设施和避难场所，并确保其处于良好状态、随时可以投入正常使用。
        3. ③加强对重点单位、重要部位和重要基础设施的安全保卫，维护社会治安秩序。
        4. ④采取必要措施，确保交通、通信、供水、排水、供电、供气、供热等公共设施的安全和正常运行。
        5. ⑤转移、疏散或者撤离易受突发事件危害的人员并予以妥善安置，转移重要财产。
        6. ⑥关闭或者限制使用易受突发事件危害的场所，控制或者限制容易导致危害扩大的公共场所的活动。
        7. ⑦及时向社会发布有关采取特定措施避免或者减轻危害的建议、劝告。
        8. ⑧法律、法规、规章规定的其他必要的防范性、保护性措施。
6. **6、突发事件的政府应急处置**：自然灾害、事故灾难或者公共卫生事件发生后，履行统一领导职责的人民政府可以采取下列一项或者多项应急处置措施：
    
    1. （1）组织营救和救治受害人员，疏散、撤离并妥善安置受到威胁的人员以及采取其他救助措施；
    2. （2）迅速控制危险源，标明危险区域，封锁危险场所，划定警戒区，实行交通管制以及其他控制措施；
    3. （3）立即抢修被损坏的交通、通信、供水、排水、供电、供气、供热等公共设施，向受到危害的人员提供避难场所和生活必需品，实施医疗救护和卫生防疫以及其他保障措施；
    4. （4）禁止或者限制使用有关设备、设施，关闭或者限制使用有关场所，中止人员密集的活动或者可能导致危害扩大的生产经营活动以及采取其他保护措施；
    5. （5）启用本级人民政府设置的财政预备费和储备的应急救援物资，必要时调用其他急需物资、设备、设施、工具；
    6. （6）组织公民参加应急救援和处置工作，要求具有特定专长的人员提供服务；
    7. （7）依法从严惩处囤积居奇、哄抬物价、制假售假等扰乱市场秩序的行为，稳定市场价格，维护市场秩序；
    8. （8）依法从严惩处哄抢财物、干扰破坏应急处置工作等扰乱社会秩序的行为，维护社会治安；
    9. （9）采取防止发生次生、衍生事件的必要措施；
    10. （10）保障食品、饮用水、燃料等基本生活必需品的供应；

**例**：多选（2017国家）如果得到集装箱冒烟的报告后，地方政府应立即发出了突发事件的二级警报，在预警期内可以采取的措施有：

1. A.启动应急预案，责令有关部门及时收集、报告有关信息
2. B.向社会公布反映突发事件信息的渠道，定时发布预测信息和评估结果
3. C.控制公司法人代表并中止该公司生产经营，防止发生次生、衍生事件
4. D.责令应急救援队伍、负有特定职责的人员进入待命状态,调集应急救援所需物资

### 应急处理

> 根据突发事件的发展过程，公安应急处理能力分为事态研判、信息上报、合理处置和善后恢复四个部分。

1. **1、事态研判**：
    
    1. （1）研判现场基本信息：包含人、事、地、时、原因、结果、趋势等。
    2. （2）研判突发事件处置所需情况：
        1. ①突发事件的类型；
        2. ②应急处置方案，包括：处置力量、处置机制、处置措施、处置物资。
2. **2、信息上报**：
    
    1. （1）信息上报的一般性规定：县级以上地方各级人民政府及其有关主管部门应当按照规定向上一级人民政府及其有关主管部门报告，必要时可以越级上报。
    2. （2）群体性事件的信息上报：公安机关处置群体性事件迅速向党委政府和上级公安机关报告，特别重大紧急情况可以越级上报，直至公安部。跨区的应当由共同上级公安机关批准。
    3. （3）恐怖事件的信息上报：国家设立反恐怖主义工作领导机构，统一领导和指挥全国反恐怖主义工作。设区的市级以上地方人民政府设立反恐怖主义工作领导机构，县级人民政府根据需要设立反恐怖主义工作领导机构，在上级反恐怖主义工作领导机构的领导和指挥下，负责本地区反恐怖主义工作。
3. **3、公安应急处置程序**：
    
    1. （1）警戒封控：公安机关及其人民警察在使用警戒带设置警戒区时，在场人员应当服从民警指令，无关人员退出警戒区；对破坏冲闯警戒带或者擅自进入警戒区的，经警告无效，可以强制带离现场，并可以依法给予处罚。
    2. （2）组织人员安全疏散
    3. （3）救助受害人
    4. （4）控制事态：对于社会安全事件，可以控制暴力攻击的实施者、目击者、知情人、被害人亲属、过于接近现场的新闻媒体记者。对于事故灾害，可以对现场的各种危险源进行控制。同时，对特定区域的建筑物、交通工具、设备、设施以及燃料、燃气、电力、水的供应进行控制。
    5. （5）保护现场：主要有固定证据；现场治安维护；保护重要设施、目标与贵重财物物品的安全；应对采访拍摄。
        1. > 应对采访拍摄时，未经授权，任何单位和民警不接受采访或公开发布事件信息。对上级或本级公安机关同意采访拍摄的，应当配合。依法处置未经同意而进行的采访拍摄活动。
            
    6. （6）现场处结
4. **4、公安机关应急处置措施**：
    
    1. （1）现场管制：
        1. ①命令现场管制区域内的所有人员在限定时间内离开现场
        2. ②确定管制现场周边设置警戒线，未经执行人员批准，任何人不得擅自出入
        3. ③查验管制现场内及进入现场人员的身份证件
        4. ④实行区域性交通管制，禁止无关车辆出入，对过往车辆进行治安检查
        5. ⑤检查嫌疑人员随身携带的物品
        6. ⑥限制有关公共场所内的活动
    2. （2）强行带离现场：强制带离现场是指公安机关在依法履行治安管理、制止违法犯罪职责时，将危害社会治安秩序或者威胁公共安全的人员，强行带离违法犯罪或突发事件现场，或者进行审查的行政强制措施。
    3. （3）强行驱散：强行驱散是指公安机关在依法履行治安管理、制止违法犯罪职责时，对危害社会治安秩序或者威胁公共安全的群体人员，采取强制手段迫使其解散并离开违法犯罪或突发事件现场的行政强制措施。
    4. （4）强制遣送：强制遣送是指公安机关在处理非法集会、游行、示威活动时，对在本人居住地以外的城市发动、组织集会游行示威，强制其返回原居住地的行政强制措施。
    5. （5）紧急征用：紧急征用是公安机关及其人民警察在紧急情况下，为满足行法定职责的需要临时向公民、单位征集、使用所需交通工具和设备、设施等物资的行政强制措施。
5. **5、善后恢复**：公安机关在突发事件善后恢复中主要的工作是:
    
    1. （1）案件调查
    2. （2）人口清查
    3. （3）秩序恢复
    4. （4）舆情引导

### 常见突发事件的应急处理

1. **1、群体性事件应急处理**：群体性事件是公众为了满足自身的利益需要，通过非法聚集、围堵等方式表达意愿，严重扰乱社会秩序，危害公共安全，侵犯公民人身安全和公私财产安全的集群行为。
    
    1. （1）统一领导原则：群体性事件发生后，公安机关要在党委和政府的统一领导下，依法维护社会秩序，平息事态，协助有关部门做好群众工作，发挥社会整体控制作用，妥善处置群体性事件。
    2. （2）三慎原则：慎用警力、慎用强制措施、慎用武器警械。
    3. （3）三可三不可原则：可散不可聚，可顺不可激，可解不可结。
    4. （4）公安机关处理群体性事件的核心工作：
        1. ①快速反应，赶赴现场
        2. ②控制局势，控制现场
        3. ③疏导教育，孤立闹事者
        4. ④开展调查，组织对话
2. **2、火灾应急处理**：
    
    1. （1）火灾时人的异常心理：
        1. 从众心理：毫无主见，人云亦云。
        2. 绝望心理：破罐破摔，听天由命。
        3. 逆反心理：失去理智，不该做的反而做了。
        4. 归巢心理：未查明火情，盲目走经常使用的走廊、楼梯、电梯、出口等。
        5. 重返心理：为抢救家人或财物，重新冲入火海
    2. （2）群众火灾自救常识
        1. 保持冷静，判明情况
        2. 利用疏散通道和安全出口自救逃生
        3. 自制器材逃生
        4. 寻找避难所逃生
        5. 在逃生过程中防止中毒
    3. （3）警察在火灾中的任务
        1. 有组织疏散
        2. 正确通报，防止混乱
        3. 疏散引导：语言引导；掩护疏散；制止脱险者重返。
        4. 禁止使用普通电梯运载人员
3. **3、突发自然灾害、事故灾害、公共卫生事件应急处理**：
    
    1. （1）向隶属的人民政府和上级公安机关履行报告职责，同时，向职能部门说明情况
